## 🎯 Betting Recommendation System - Добавлено

### Что добавлено:

**1. Обновленные System Prompts (bot.py, Lines 204-330)**
- Все 11 DISCIPLINE_PROMPTS теперь включают обязательный формат вывода с вероятностью и рекомендацией по ставке
- LLM теперь явно инструктирует указывать вероятность в формате: `📊 **Вероятность победы (1-я команда):** X%`

**2. Новый модуль: `services/betting_calculator.py`**

Функции:
- `extract_probability(llm_response)` - Извлекает % вероятности из текста LLM
  - Ищет паттерны: "Вероятность...: X%", "P = X%", "вероятность: X%", "X%"
  - Возвращает: float или None

- `calculate_stake_percentage(probability)` - Вычисляет размер ставки по формуле:
  - P > 80% → **6%** от банка (Ультра-уверенность)
  - 66% ≤ P ≤ 80% → **3%** от банка (Высокая вероятность)
  - 55% ≤ P ≤ 65% → **1%** от банка (Рискованно/Валуй)
  - P < 55% → **None** (⚠️ ПРОПУСТИТЬ)

- `format_stake_recommendation(probability, stake_percent)` - Форматирует рекомендацию
  - Возвращает красиво отформатированную строку с рекомендацией

- `get_bet_recommendation(llm_response)` - Главная функция (полный цикл)
  - Принимает: тек ст ответа от LLM
  - Возвращает: (probability, recommendation_string)

**3. Интеграция в bot.py (Lines 856-902)**
- `start_analysis()` функция теперь:
  1. Получает ответ от LLM (как раньше)
  2. Вызывает `get_bet_recommendation()` 
  3. Если вероятность найдена → добавляет рекомендацию в конец сообщения
  4. Отправляет расширенный ответ пользователю

### Пример работы:

**Input:** LLM вернул анализ матча с вероятностью 75%

**Output:**
```
[Анализ матча...]

📊 **ПРОГНОЗ:** Team A vs Team B

[Детали анализа...]

💰 **РЕКОМЕНДАЦИЯ:** 3% от вашего банка (вероятность 75% = Высокая вероятность)
```

### Как это работает вместе:

1. **Пользователь** → выбирает дисциплину, матч, дату
2. **Bot** → собирает данные через Fetcher-ы
3. **Groq** → получает инструкцию с новым форматом, возвращает анализ + вероятность
4. **Betting Calculator** → парсит вероятность, вычисляет ставку, форматирует рекомендацию
5. **User** → получает анализ + рекомендацию по размеру ставки ✅
